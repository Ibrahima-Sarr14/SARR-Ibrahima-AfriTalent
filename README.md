## 📋 Description
AfriTalent est une plateforme fictive qui connecte les freelances africains aux entreprises du monde entier. Ce projet a été réalisé dans le cadre du cours de développement web à l'ISI Dakar.

---

## 🛠️ Technologies utilisées
- HTML5 — Structure des pages
- CSS3 — Mise en forme et animations
- Bootstrap 5.3 — Framework CSS responsive
- JavaScript Vanilla — Interactivité
- Git & GitHub — Versionnage et déploiement

---

## ✨ Fonctionnalités principales
- Navbar responsive avec dark mode (localStorage)
- Hero avec image d'arrière-plan Afrique
- Compteurs animés au scroll (IntersectionObserver)
- Filtrage dynamique des freelances sans rechargement
- Validation du formulaire de contact avec regex
- Bouton retour en haut animé
- Carousel de témoignages Bootstrap
- Bento Grid pour les étapes
- Design responsive (mobile, tablette, desktop)
